# 1. True Error vs Empirical Error

- True Error đánh giá performance cả tập train và tập test
- Empirical Error: Chỉ đánh giá tập train

# 2. Overfiting & Underfiting

- Overfiting: When the model have too many features, the model might fit the training data very well but fail to generalize new example.
- Underfiting: When the model have too few features to generalize the new example.
